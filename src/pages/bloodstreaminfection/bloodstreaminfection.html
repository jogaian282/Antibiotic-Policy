Blood Stream Infection

Type I
  • Ceftriaxone / cefotaxime (For Non - ESBL suspect)
  • or
  • ampicillin / sulbactum
Type II
  • Piperacillin / Tazobactam +/- Amikacin
  • (or)
  • Cefoperazone / Sulbactam +/- Amikacin
Type III
  • Meropenem + Amikacin
  • (or)
  • Imipenem + Amikacin
  • (or) cefepime (last resort reserve group)
Type IV
  • Based on IDSA Guildelines and risk factors empirical antifungal
  • Flucanazole can be added if candida infection is suspected in Febrile Neutropenic patients
NOTE :

Renal Function to be assessed if INJ.Amikacin is to be continued

Vancomycin or Linezolid if MRSA is suspected

Metronidazole can be added if anaerobic cover of antibiotics is not adequate
